# pedestrian_fall_detection
自动扶梯乘客异常行为识别项目，由于项目合作规定，没有上传模型训练以及逆行检测模块的代码，仅上传了使用瑞芯微rk3399进行行人摔倒检测的部分代码，检测行人并判断是否摔倒。
